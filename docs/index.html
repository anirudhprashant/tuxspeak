How it works

Four steps, no cloud dashboard, no daemon lock-in.

01

Alt+Space

Starts recording. Indicator shows (recording...).

02

Speak

Groq Whisper transcribes in real time.

03

Alt+Space again

Gemini 3.1 Flash Lite polishes grammar and cuts filler words.

04

Typed at cursor

Result lands via uinput. Silence cancels; nothing pastes.

Features

Local-first, cross-app, open source.

Cross-app dictation

Works in browser, terminal, IDE, chat, email — any text field, any window.

AI polishing

Gemini 3.1 Flash Lite cleans up filler words and grammar before typing.

Dictation history

Every dictation saved with timestamp, raw and polished versions, re-copyable.

Local-first

Audio stays on your machine. Only the audio blob leaves, for transcription and polishing.

Open source

GPL 3.0 for the wrapper, MIT for the xhisper core it's built on.

Free APIs

Groq Whisper and Gemini both ship generous free tiers. No credit card needed.

Install

One script. Handles deps, keys, build, and the shortcut.

# clone and run the installer
git clone https://github.com/anirudhprashant/tuxspeak.git
cd tuxspeak
./install-xhisper.sh
ServiceWhat it doesFree key
GroqSpeech-to-text (Whisper)console.groq.com/keys
GeminiText polishing (Gemini 3.1 Flash Lite)aistudio.google.com/apikey

Press Alt+Space. Speak. Done.

Requires Linux + PipeWire. Reboot after install if you were added to the input group.
